# Flow Studio — 项目概述与技术架构

> AI 可视化工作流编辑器，类似 ComfyUI 的节点式操作体验

---

## 一、产品定位

一个可视化的 AI 工作流编辑器，用户通过拖拽节点、连线组合的方式，自定义 AI 任务的执行流程。支持对话、图像生成、视频生成、网页搜索等多种 AI 能力的编排。

## 二、核心用户场景

```
场景1：内容创作者
  搜索热点 → AI 总结 → AI 润色 → 输出文章

场景2：设计师
  文本提示词 → AI 扩写提示词 → 图像生成 → 输出图片

场景3：开发者
  代码问题描述 → AI 对话 → 代码审查 → 输出建议

场景4：视频创作者
  文本描述 → AI 生成脚本 → 视频生成 → 输出视频
```

## 三、参考产品

- ComfyUI（最新版）— 核心交互和视觉风格
- Dify — 工作流编排思路
- Node-RED — 节点连线和数据传递

## 六、为什么需要后端执行

```
纯前端方案的问题：
  ❌ API Key 暴露在浏览器中，不安全
  ❌ 关闭浏览器后执行中断
  ❌ 大量图片/视频数据占用浏览器内存
  ❌ 无法定时/后台执行工作流

后端执行的好处：
  ✅ API Key 安全存储在服务端
  ✅ 关闭浏览器后继续执行
  ✅ 文件直接保存到磁盘
  ✅ 未来可扩展定时任务、API 对外服务
```
